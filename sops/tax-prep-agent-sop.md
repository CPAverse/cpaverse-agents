## 1. OVERVIEW

### 1.1 Purpose

This document defines the standard operating procedures for the CPAverse Tax Prep Agent — an autonomous AI agent that supports the tax preparation workflow at Josh Mauer CPA LLC.

### 1.2 Firm Profile

- **Firm:** Josh Mauer CPA LLC, Overland Park, Kansas
- **Founder:** Josh Mauer, CPA (nearly 30 years experience)
- **Staff:** 6-7 full-time employees
- **Annual Revenue:** ~$900K
- **Technology Company:** CPAverse Inc. (Delaware C-corporation, NAICS 511210)

### 1.3 Agent Design Philosophy

The agent learns systems the way a new employee would — by interacting with the software, referencing online resources, and building knowledge with each return processed. It does not follow brittle step-by-step screen instructions. It adapts.
